Dataset: Procurement KPI Analysis Dataset
This dataset is used to analyze procurement performance at the purchase-order level. It supports KPI tracking for delivery lead time, supplier performance, cost savings, and quality compliance in Power BI.

Key Fields
1)Order_Date, Delivery_Date → Lead time analysis
2)Days_For_Delivery → Calculated delivery duration
3)Supplier, Item_Category → Performance segmentation
4)Unit_Price, Negotiated_Price, Quantity → Cost savings
5)Defective_Units, Compliance → Quality metrics

Key KPIs
1)Average & percentile-based delivery days (Q1 / Median / Q3)
2)Supplier compliance and defect rate
3)Total spend and cost savings

Dynamic Analysis
1)Quartiles are calculated using Power BI measures and respond to all report filters (Supplier, Category, Date).
2)Use Case: Identify delivery bottlenecks, benchmark supplier SLAs, and improve procurement efficiency.
